
# genesis_services

Manage services in the Genesis installation

## Usage

```console
                                                                                
 Usage: genesis services [OPTIONS] COMMAND [ARGS]...                            
                                                                                
```

## Options

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
