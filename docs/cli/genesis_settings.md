
# genesis_settings

Modify genesis settings files

## Usage

```console
                                                                                
 Usage: genesis settings [OPTIONS] COMMAND [ARGS]...                            
                                                                                
```

## Options

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
