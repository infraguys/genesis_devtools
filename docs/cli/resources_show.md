
# resources_show

Show resource

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis resources show [OPTIONS] RESOURCE_NAME_UUID                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                                                                           
```

## Options

* `element`:
    * Type: text
    * Default: `none`
    * Usage: `-e
--element`

  Name or uuid of the element

* `resource_name_uuid` (REQUIRED):
    * Type: text
    * Default: `sentinel.unset`
    * Usage: `resource_name_uuid`

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
