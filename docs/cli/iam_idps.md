## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis iam idps [OPTIONS] COMMAND [ARGS]...                                                                                                                                                                                                                                                       
                                                                                                                                                                                                                                                                                                           
```

    * Default: `false`
    * Usage: `--help`
