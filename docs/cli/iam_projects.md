
# iam_projects

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis iam projects [OPTIONS] COMMAND [ARGS]...                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                                                           
```

    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
