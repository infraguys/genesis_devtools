
# realms_ssh

Connect to genesis realm

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis realms ssh [OPTIONS]                                                                                                                                                                                                                                                                       
                                                                                                                                                                                                                                                                                                           
```

## Options

* `realm`:
    * Type: text
    * Default: `none`
    * Usage: `-r
--realm`

  Realm to connect to

* `username`:
    * Type: text
    * Default: `ubuntu`
    * Usage: `-u
--username`

  Default username

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
