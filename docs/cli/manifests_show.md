
# manifests_show

Show manifest general information

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis manifests show [OPTIONS] NAME                                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                                                                                           
```

## Options

* `name` (REQUIRED):
    * Type: text
    * Default: `sentinel.unset`
    * Usage: `name`

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
