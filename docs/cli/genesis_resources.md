
# genesis_resources

Manage resources in the Genesis installation

## Usage

```console
Usage: genesis resources [OPTIONS] COMMAND [ARGS]...
```

## Options

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.

## CLI Help

```console
Usage: genesis resources [OPTIONS] COMMAND [ARGS]...

  Manage resources in the Genesis installation

Options:
  --help  Show this message and exit.

Commands:
  list  List resources
  show  Show resource
```
