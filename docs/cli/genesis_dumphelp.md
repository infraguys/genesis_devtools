
# genesis_dumphelp

tool for creating docs files for cli commands

## Usage

```console
                                                                                
 Usage: genesis dumphelp [OPTIONS]                                              
                                                                                
```

## Options

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
