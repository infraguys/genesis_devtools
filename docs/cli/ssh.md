
# ssh

Connect to genesis stand/element

## Usage

```console
Usage: genesis ssh [OPTIONS]
```

## Options

*   `stand`:
    *   Type: STRING
    *   Default: `none`
    *   Usage: `-s
--stand`

  Stand to connect to

*   `username`:
    *   Type: STRING
    *   Default: `ubuntu`
    *   Usage: `-u
--username`

  Default username

*   `help`:
    * Type: BOOL
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.

## CLI Help

```console
Usage: genesis ssh [OPTIONS]

  Connect to genesis stand/element

Options:
  -s, --stand TEXT     Stand to connect to
  -u, --username TEXT  Default username
  --help               Show this message and exit.
```
