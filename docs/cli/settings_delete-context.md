
# settings_delete-context

Delete the specified context from the settings

## Usage

```console
                                                                                
 Usage: genesis settings delete-context [OPTIONS] NAME                          
                                                                                
```

## Options

* `name` (REQUIRED):
    * Type: text
    * Default: `sentinel.unset`
    * Usage: `name`

* `realm`:
    * Type: text
    * Default: `sentinel.unset`
    * Usage: `-r
--realm`

  Name of the realm

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
