
# genesis_openapi

tool for creating openapi spec files

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis openapi [OPTIONS] [PATH]                                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                                                           
```

## Options

* `url`:
    * Type: text
    * Default: `none`
    * Usage: `-u
--url`

  openapi url

* `endpoint`:
    * Type: text
    * Default: `none`
    * Usage: `-e
--endpoint`

* `path`:
    * Type: file
    * Default: `sentinel.unset`
    * Usage: `path`

  Path to target file

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
