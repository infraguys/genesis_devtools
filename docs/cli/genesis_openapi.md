
# genesis_openapi

tool for creating openapi spec files

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis openapi [OPTIONS] PATH                                                                                                                                                                                                                                                                     
                                                                                                                                                                                                                                                                                                           
```

## Options

* `path` (REQUIRED):
    * Type: file
    * Default: `sentinel.unset`
    * Usage: `path`

  Path to target file

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
