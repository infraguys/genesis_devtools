```console
 Usage: genesis autocomplete [OPTIONS]                                                                                                                                                                                                                                                                     
                                                                                                                                                                                                                                                                                                           
```

## Options

* `shell`:
    * Type: choice
    * Default: `none`
    * Usage: `-s
--shell`

  shell kind

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
