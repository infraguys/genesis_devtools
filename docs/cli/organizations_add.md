
# organizations_add

Add a new organization to the Genesis installation

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis iam organizations add [OPTIONS]                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                                                                           
```

## Options

* `uuid`:
    * Type: uuid
    * Default: `none`
    * Usage: `-u
--uuid`

  UUID of the organization

* `name`:
    * Type: text
    * Default: `test_name`
    * Usage: `-n
--name`

  Name of the organization

* `description`:
    * Type: text
    * Default: ``
    * Usage: `-D
--description`

  Description of the organization

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
