
# exports_show

Show export

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis e exports show [OPTIONS] NAME_UUID                                                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                                                           
```

## Options

* `element`:
    * Type: text
    * Default: `none`
    * Usage: `-e
--element`

  Name or uuid of the element

* `name_uuid` (REQUIRED):
    * Type: text
    * Default: `sentinel.unset`
    * Usage: `name_uuid`

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
