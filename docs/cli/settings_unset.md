
# settings_unset

Unset an individual value in a settings file

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis settings unset [OPTIONS] KEY                                                                                                                                                                                                                                                               
                                                                                                                                                                                                                                                                                                           
```

## Options

* `key` (REQUIRED):
    * Type: text
    * Default: `sentinel.unset`
    * Usage: `key`

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
