
# settings_delete-realm

Delete the specified realm from the settings

## Usage

```console
                                                                                
 Usage: genesis settings delete-realm [OPTIONS] REALM                           
                                                                                
```

## Options

* `realm` (REQUIRED):
    * Type: text
    * Default: `sentinel.unset`
    * Usage: `realm`

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
