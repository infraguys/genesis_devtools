
# genesis_configs

Manage configs in the Genesis installation

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis configs [OPTIONS] COMMAND [ARGS]...                                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                                                                           
```

## Options

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
