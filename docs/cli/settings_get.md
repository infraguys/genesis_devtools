
# settings_get

Get an individual value from a settings file

## Usage

```console
                                                                                
 Usage: genesis settings get [OPTIONS] KEY                                      
                                                                                
```

## Options

* `key` (REQUIRED):
    * Type: text
    * Default: `sentinel.unset`
    * Usage: `key`

* `help`:
    * Type: boolean
    * Default: `false`
