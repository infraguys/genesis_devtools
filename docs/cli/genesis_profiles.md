
# genesis_profiles

Manage profiles in the Genesis installation

## Usage

```console
                                                                                
 Usage: genesis profiles [OPTIONS] COMMAND [ARGS]...                            
                                                                                
```

## Options

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
