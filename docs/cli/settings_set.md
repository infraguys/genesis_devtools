
# settings_set

Set an individual value in a settings file

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis settings set [OPTIONS] KEY VALUE                                                                                                                                                                                                                                                           
                                                                                                                                                                                                                                                                                                           
```

## Options

* `key` (REQUIRED):
    * Type: text
    * Default: `sentinel.unset`
    * Usage: `key`

* `value` (REQUIRED):
    * Type: text
    * Default: `sentinel.unset`
    * Usage: `value`

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
