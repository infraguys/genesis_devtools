
# iam_permissions

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis iam permissions [OPTIONS] COMMAND [ARGS]...                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                                                                           
```

## Options

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
