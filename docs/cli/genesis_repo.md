
# genesis_repo

Manager Genesis repository

## Usage

```console
Usage: genesis repo [OPTIONS] COMMAND [ARGS]...
```

## Options

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
