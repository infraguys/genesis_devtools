
# hypervisors_init

Initialize hypervisor

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis compute hypervisors init [OPTIONS]                                                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                                                           
```

## Options

* `romfile_version`:
    * Type: text
    * Default: `latest`
    * Usage: `--romfile_version`

  version of the rom file

* `pool_name`:
    * Type: text
    * Default: `latest`
    * Usage: `--pool_name`

  storage pool name

* `packer`:
    * Type: boolean
    * Default: `true`
    * Usage: `-p
--packer`

  Install packer

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
