
# compute_nodes

Manage nodes in the Genesis installation

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis compute nodes [OPTIONS] COMMAND [ARGS]...                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                           
```

## Options

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
