
# genesis_auth

Authenticate and manage IAM token

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis auth [OPTIONS] COMMAND [ARGS]...                                                                                                                                                                                                                                                           
                                                                                                                                                                                                                                                                                                           
```

## Options

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
