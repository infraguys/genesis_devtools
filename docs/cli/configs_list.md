
# configs_list

List configs

## Usage

```console
Usage: genesis configs list [OPTIONS]
```

## Options

* `node`:
    * Type: uuid
    * Default: `none`
    * Usage: `-n
--node`

  Filter configs by node

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
