
# manifests_validate

Validate manifest

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis manifests validate [OPTIONS] PATH_OR_NAME                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                           
```

## Options

* `repository`:
    * Type: text
    * Default: `https://repository.genesis-core.tech/genesis-elements/`
    * Usage: `-r
--repository`

  Repository endpoint

* `path_or_name` (REQUIRED):
    * Type: text
    * Default: `sentinel.unset`
    * Usage: `path_or_name`

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
