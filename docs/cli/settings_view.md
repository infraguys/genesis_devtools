
# settings_view

Display merged genesis settings

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis settings view [OPTIONS]                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                           
```

## Options

* `raw`:
    * Type: boolean
    * Default: `false`
    * Usage: `--raw`

  Display the raw merge of the genesis config

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
