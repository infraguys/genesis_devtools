
# genesis_get-version

Return the version of the project

## Usage

```console
                                                                                
 Usage: genesis get-version [OPTIONS] ELEMENT_DIR                               
                                                                                
```

## Options

* `element_dir` (REQUIRED):
    * Type: path
    * Default: `sentinel.unset`
    * Usage: `element_dir`

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
