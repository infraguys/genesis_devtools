
# nodes_list

List nodes

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis compute nodes list [OPTIONS]                                                                                                                                                                                                                                                               
                                                                                                                                                                                                                                                                                                           
```

## Options

* `project_id`:
    * Type: text
    * Default: `none`
    * Usage: `-P
--project-id`

  Filter nodes by project

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
