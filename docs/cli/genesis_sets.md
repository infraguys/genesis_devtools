
# genesis_sets

Manage sets in the Genesis installation

## Usage

```console
                                                                                
 Usage: genesis sets [OPTIONS] COMMAND [ARGS]...                                
                                                                                
```

## Options

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
