
# elements_uninstall

Uninstall element by UUID, path or name

## Usage

```console
Usage: genesis elements uninstall [OPTIONS] PATH_UUID_NAME
```

## Options

* `path_uuid_name` (REQUIRED):
    * Type: text
    * Default: `sentinel.unset`
    * Usage: `path_uuid_name`

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
