
# genesis_ssh

Connect to genesis stand/element

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis ssh [OPTIONS]                                                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                                                                                           
```

## Options

* `stand`:
    * Type: text
    * Default: `none`
    * Usage: `-s
--stand`

  Stand to connect to

* `username`:
    * Type: text
    * Default: `ubuntu`
    * Usage: `-u
--username`

  Default username

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
