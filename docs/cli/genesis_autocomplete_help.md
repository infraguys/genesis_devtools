
# genesis_autocomplete_help

Display a autocomplete help

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis autocomplete_help [OPTIONS]                                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                                                                           
```

## Options

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
