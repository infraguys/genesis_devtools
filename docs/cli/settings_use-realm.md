
# settings_use-realm

Set the current-realm in a settings file

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis settings use-realm [OPTIONS] REALM                                                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                                                           
```

## Options

* `realm` (REQUIRED):
    * Type: text
    * Default: `sentinel.unset`
    * Usage: `realm`

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
