
# genesis_projects

Manage projects in the Genesis installation

## Usage

```console
                                                                                
 Usage: genesis projects [OPTIONS] COMMAND [ARGS]...                            
                                                                                
```

## Options

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
