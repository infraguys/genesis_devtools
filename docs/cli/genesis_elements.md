
# genesis_elements

Manage elements in the Genesis installation

## Usage

```console
                                                                                                                                                                                                                                                                                                           
 Usage: genesis elements [OPTIONS] COMMAND [ARGS]...                                                                                                                                                                                                                                                       
                                                                                                                                                                                                                                                                                                           
```

## Options

* `help`:
    * Type: boolean
    * Default: `false`
    * Usage: `--help`

  Show this message and exit.
